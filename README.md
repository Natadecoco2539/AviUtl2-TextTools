# AviUtl2-TextTools

AviUtl2用 テキスト編集＆分割プラグインセット v1.0
作成者: [なたでここ/Natadecoco_2539]

■ 概要
AviUtl2でテキストオブジェクトを便利に扱うためのプラグイン2種セットです。
ダークテーマ・フラットデザインで、AviUtl2の見た目に馴染みます。

1. TextEditor.aux2 (文字別詳細エディタ)
   テキストの文字ごとに「サイズ・フォント・太字・斜体・色」を個別に設定できます。
   設定保存機能付き。

2. TextSplitter.aux2 (テキスト分割)
   テキストオブジェクトを1文字ごとのオブジェクトに分割して配置します。
   簡単な操作で分割できます。

■ インストール方法
このフォルダに入っている「TextEditor.aux2」と「TextSplitter.aux2」を
AviUtl2の「Plugin」フォルダに入れてください。

■ 使い方
[TextEditor]
メニューの「表示」→「文字別詳細エディタ」を選択してウィンドウを表示します。
テキストオブジェクトを選択し、ウィンドウ内のボタンで操作してください。

[TextSplitter]
メニューの「表示」→「テキスト分割」を選択してウィンドウを表示します。
テキストオブジェクトを選択し、「テキスト分割」ボタンを押してください。
